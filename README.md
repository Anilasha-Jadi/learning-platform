\# Learning Platform CMS – Phase 1



This project is a CMS for managing Programs, Terms and Lessons.



\## What is implemented



\- Django backend

\- PostgreSQL database

\- Docker Compose setup

\- Program → Term → Lesson data model

\- Slug-based navigation

\- Django Admin for content management

\- CMS UI for browsing programs, terms and lessons



\## Tech Stack



\- Django

\- PostgreSQL

\- Docker \& Docker Compose



\## How to run locally



1\. Install Docker Desktop

2\. Run:



docker compose up --build



3\. Open:

http://localhost:8000



4\. Admin:

http://localhost:8000/admin



\## Status



This is Phase 1 of the full CMS.

Publishing workflow, worker, and public catalog API

will be added in next phases.
